星函云代码隐私政策

一、政策前言

“星函云代码”（以下简称“本应用”）高度重视用户隐私保护，本隐私政策旨在向您说明本应用收集、使用、存储、保护用户信息的规则，以及您享有的相关权利。您使用本应用即视为同意本隐私政策的全部内容。

开发者有权根据法律法规及业务调整更新本政策，更新后将通过本应用内公告予以通知，更新后的政策自通知之日起生效。

二、适用范围

本隐私政策适用于您使用本应用提供的所有服务，包括代码编辑、文件读写、代码编译运行等功能过程中的信息处理行为。

三、信息收集与使用

（一）收集的信息类型

本应用仅为实现核心功能收集必要信息，不主动收集您的姓名、手机号、身份证号等个人身份信息，具体收集的信息如下：

  • 1. 设备权限相关信息（仅在您授权后获取）
    • 网络权限：用于将您编辑的代码提交至第三方 SDK 服务、获取编译运行结果，以及核查网络状态；
    • 网络状态信息：用于判断设备当前网络环境（如WiFi/移动网络），优化代码提交策略，避免不必要的网络消耗；
    • 剪切板内容：仅在您主动粘贴代码时读取，用于实现代码编辑功能，读取后仅在本地使用，不存储、不上传；
    • 文件读取权限：仅读取您主动选择的文件内容（限代码文件），用于在本应用中编辑，不读取其他无关文件；
    • 文件保存权限：仅将您编辑后的代码文件保存至您指定的本地存储位置，不擅自保存其他文件。
  • 2. 代码内容

    您提交编译的代码内容，仅会发送至第三方 SDK 服务用于编译运行，开发者不会存储您的代码内容，编译完成后第三方 SDK 服务的信息处理规则由其自行负责。

（二）信息使用目的

收集的信息仅用于实现本应用的核心功能，具体包括：

  • 为您提供稳定的代码编辑环境；
  • 将您的代码提交至第三方 SDK 服务完成编译运行，并返回结果；
  • 优化本应用的使用体验（如根据网络状态调整代码提交策略）；
  • 排查并解决本应用运行过程中的故障。

四、信息存储与保护

1. 本应用收集的所有信息（除提交至第三方 SDK 服务的代码外）均仅存储在您的本地设备中，开发者不会上传、存储您的任何本地信息；

2. 开发者采取合理的技术措施（如本地数据加密）保护您的本地信息安全，但因设备被ROOT/越狱、恶意软件攻击等非开发者原因导致的信息泄露，开发者不承担责任；

3. 您可随时删除本地存储的代码文件、清空剪切板，自行管理本地信息。

五、第三方 SDK 服务说明

1. 本应用的代码编译运行功能依赖第三方 SDK 服务实现，仅向该第三方提交您的代码内容，不提交任何其他信息；

2. 第三方 SDK 服务的隐私政策由其独立制定，开发者无法控制第三方的信息处理行为，建议您自行查阅第三方的隐私政策；

3. 开发者不对第三方 SDK 服务的信息安全、服务质量承担任何责任。

六、信息共享与披露

除以下情况外，开发者不会将您的任何信息共享、转让或披露给任何第三方：

  • 经您明确授权或同意；
  • 法律法规、司法机关或行政机关的强制性要求；
  • 为保护开发者、用户或公众的合法权益，在紧急情况下的必要披露。

七、用户权利

1. 您可在设备的“设置-应用管理”中自主开启或关闭本应用的各项权限，关闭权限可能导致部分功能无法使用；

2. 您可随时删除本应用中编辑的代码、本地保存的文件，清空相关信息；

3. 您可随时卸载本应用，卸载后本地存储的相关信息将由您自行决定是否保留；

4. 如您对信息处理有疑问，可通过开发者联系方式咨询、投诉。

八、未成年人保护

本应用主要面向具备代码编辑能力的用户，如未成年人使用本应用，需在监护人的指导下进行，并由监护人承担相应责任。开发者不会主动收集未成年人的个人信息，如发现未成年人信息，将采取措施保护其隐私。

九、隐私政策的变更

开发者可根据法律法规更新、业务调整等情况修改本隐私政策，修改后将通过本应用内弹窗、公告等方式通知您，您继续使用本应用即视为同意修改后的政策。
